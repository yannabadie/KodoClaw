description: Show current Kodo state (mode, autonomy, plan, memory, cost)

Show the current Kodo status including:
1. Active mode and autonomy level
2. Memory stats (MemCells count, MemScenes count)
3. Active plan progress (if any)
4. Session and daily token cost
5. Any active anomaly alerts

Format as the inline dashboard box.
