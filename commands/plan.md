---
name: kodo:plan
description: View or manage the milestone roadmap
args: [show|create|complete <milestone>]
---

Manage the hierarchical milestone plan:
- `show` (default) — Display current plan with progress
- `create` — Start a new milestone plan for the current task
- `complete <milestone>` — Mark a milestone as done

Show milestones with status indicators and step-wise hints for the active milestone.
